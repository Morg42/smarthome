Beschreibung
============

Plugin zur Anzeige von Informationen von ESPHome Devices


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.10**

=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


param1
======


Demo Parameter

 - Datentyp: **str**


param2
======


Demo Parameter mit Gültigkeitsliste und Standardwert

 - Datentyp: **str**
 - Standardwert: **value2**

 - Mögliche Werte:

   - **value1**
   - **value2**
   - **value3**
   - **value4**
   - **value5**


pause_item
==========


Item, um die Ausführung des Plugins zu steuern

 - Datentyp: **str**
