Beschreibung
============

Interface plugin fuer Casambi lokal via Bluetooth


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.7**
 - Minimum Python Version: **3.11**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **casambi_bt** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


mac
===


MAC-Adresse (12-stelliger hex Wert, z.B. AA:BB:CD:EF:12:34)

 - Datentyp: **str**


password
========


Kennwort des lokalen Casmbi Netzwerks

 - Datentyp: **str**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


casambi_bt_id
=============


Casambi Geraete ID

 - Datentyp: **int**

 - Minimalwert: **0**

casambi_bt_rx_key
=================


Key fuer den Empfang von Statusinformationen

 - Datentyp: **str**

 - Mögliche Werte:

   - **ON**
   - **DIMMER**
   - **VERTICAL**
   - **CCT**
   - **BACKEND_ONLINE_STAT**


casambi_bt_tx_key
=================


Key fuer das Senden von Kommandos ins Casambi Netzwerk

 - Datentyp: **str**

 - Mögliche Werte:

   - **ON**
   - **DIMMER**
   - **VERTICAL**
   - **CCT**
